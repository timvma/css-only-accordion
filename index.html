How to Build a CSS-only Accordion for Your Website

To create an accordion with CSS only, you need to use the :checked pseudo-class and the adjacent sibling combinator (+). First, you need to create a hidden checkbox for each accordion item. Then, you need to create a label element that acts as the toggle button for the checkbox. Next, you need to create a div element that contains the content of the accordion item. Finally, you need to style the label and the div elements with CSS, using the :checked pseudo-class and the adjacent sibling combinator (+) to show or hide the content when the checkbox is checked or unchecked.

HTML


<div class="accordion">
    <label class="accordion__head" for="css_only_accordion__basic"> CSS ONLY </label>
    <input id="css_only_accordion__basic" type="checkbox" class="accordion__status">
     <div class="accordion__content">
         <div class="accrodion__wrap">
            <div class="accordion__padding">
                 <p> Natus delectus qui
                    voluptatibus quod voluptatem?
                    Optio, dolor. Quasi sit incidunt 
                    atque similique ea architecto quis
                    quisquam animi, aliquam qui eligendi 
                    corporis! </p>
            </div>
        </div>
    </div>
</div>

CSS


    .accordion__status {
        display: none;
    }
    
    .accordion__content {
        display: grid;
        grid-template-rows: 0fr;
        overflow: hidden;
        transition: all ease 500ms;
    }
    
    .accordion__status:checked ~ .accordion__content {
        grid-template-rows:1fr;
        transition: all ease 250ms;
    }
    
    .accrodion__wrap {
        min-height: 0;
    }
